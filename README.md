# devops-roast Homebrew Tap

Homebrew formulae for [devops-roast](https://github.com/devops-roast) CLI tools.

## Usage

```sh
brew tap devops-roast/tap
brew install terramate-atlantis-config
```

## Available formulae

| Formula | Description |
|---------|-------------|
| [terramate-atlantis-config](https://github.com/devops-roast/terramate-atlantis-config) | Generate Atlantis config for Terramate projects |
